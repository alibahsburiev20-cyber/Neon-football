# NEON CIRCUIT FC — Turnament Backend (Cloudflare Worker + D1)

Бэкенд для еженедельного онлайн-турнира. Без него игра тоже работает —
турнир просто уходит в локальный офлайн-режим (таблица лидеров только
на этом устройстве). После деплоя всё само переключится на онлайн.

## 1. Установка wrangler (один раз, в Termux)

```bash
npm install -g wrangler
wrangler login
```

`wrangler login` откроет ссылку — авторизуй Cloudflare-аккаунт (тот же,
где уже подключены Workers/Supabase).

## 2. Создать базу D1 (один раз)

```bash
cd server
wrangler d1 create neon-circuit-fc-db
```

Команда выведет что-то вроде:

```
database_id = "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
```

Скопируй этот `database_id` и вставь в `wrangler.toml` вместо
`REPLACE_WITH_YOUR_DATABASE_ID`.

## 3. Накатить схему

```bash
wrangler d1 execute neon-circuit-fc-db --file=./schema.sql --remote
```

(Воркер и сам создаст таблицу при первом запросе, если забыть этот шаг —
но лучше сделать сразу.)

## 4. Задеплоить воркер

```bash
wrangler deploy
```

В конце команда покажет URL вида:

```
https://neon-circuit-fc-tournament.<твой-сабдомен>.workers.dev
```

## 5. Подключить URL к игре

Открой игру → Настройки → «URL турнирного сервера» → вставь URL из шага 4
(без слэша на конце) → сохранить. Всё, турнир онлайн.

Либо через консоль браузера:
```js
Tournament.setBackendUrl('https://neon-circuit-fc-tournament.xxx.workers.dev');
```

## Проверка

```bash
curl https://neon-circuit-fc-tournament.xxx.workers.dev/api/health
# {"ok":true,"period":"2026-W27"}
```

## Что учтено

- Хранится только лучший результат каждого игрока за неделю (ISO-неделя,
  считается сервером — клиент не может её подделать).
- Грубая защита от спама: не больше 200 сабмитов на игрока в неделю.
- Проверка на адекватность очков (0 ≤ score ≤ 100000) — это не полноценная
  анти-чит защита (очки всё ещё считаются на клиенте), но отсекает
  случайные и совсем наглые значения. Для полной защиты от читеров в
  будущем понадобится серверная валидация повторов матча — вне рамок MVP.
- CORS открыт всем источникам, чтобы игра работала как Telegram Mini App
  с любого домена.

## Стоимость

Бесплатный тариф Workers — 100 000 запросов/день, D1 — 5 ГБ хранилища.
Для турнира на несколько тысяч игроков в неделю этого более чем достаточно
и он останется полностью бесплатным.
