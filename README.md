## Что нового

**Поле и состав**
- Поле увеличено (1780×1000 вместо 1400×820), формат 6х6 вместо 4х4:
  вратарь, 2 защитника, 2 полузащитника, нападающий (формация 1-2-2-1).
- Экран **СОСТАВ** — выбор своей позиции на поле (вратарь / защитник /
  полузащитник / нападающий) и предпросмотр экипированного скина.
- **Автовыбор игрока у мяча**: управление само переключается на
  ближайшего к мячу партнёра, когда ты не владеешь мячом (чтобы не терять
  дриблинг во время атаки). Можно выключить в Настройках, плюс есть
  ручное переключение по кругу — клавиша `Q` или кнопка `⇄` на тачскрине.

**Прогрессия и магазин**
- Профиль игрока: уровень, опыт, DATA SHARDS (мягкая валюта) и
  OC-КРЕДИТЫ (премиум-валюта — задел под пополнение через TON Connect/
  GRAM, сейчас кнопка "Пополнить" — честная заглушка).
- Прокачка 4 характеристик (скорость/сила удара/разгон/дэш) до 5 уровня
  за шарды — применяется ко всей домашней команде.
- 11 скинов разной редкости: часть открывается уровнем, часть — за
  шарды, один легендарный ("OVERCLOCK PRISM", переливается) — только за
  премиум-валюту.
- Ежедневный бонус с серией дней, 6 достижений с наградами.
- Экран **ПРОФИЛЬ** — уровень/опыт/статистика карьеры/достижения.

**Лига**
- Экран **ВЫБОР ЛИГИ**: 3 отдельные лестницы по 5 соперников (БРОНЗА /
  СЕРЕБРО / ЗОЛОТО), прогресс по каждой хранится отдельно.
- Название лиги можно переименовать — отображается в меню и в HUD матча.

**Онлайн-турнир**
- Новый режим "ОНЛАЙН ТУРНИР": заезд на очки против системного
  соперника, результат = голы×100 + лучшая серия×10 + овердрайв-удары×15.
- Таблица лидеров — еженедельная (неделя считается на сервере, её нельзя
  подделать с клиента). Пока сервер не задеплоен, работает в **локальном
  офлайн-режиме** (таблица только на этом устройстве) — экран не
  "мёртвый", просто без интернет-соревнования.
- Бэкенд — Cloudflare Worker + D1, лежит в `/server`. Как задеплоить
  (5 команд в Termux) — `/server/README.md`. Это то, что уже реально
  протестировано логически (см. ниже), но задеплоить может только сам
  Tamy — у меня нет сетевого доступа к Cloudflare из песочницы.

**Адаптивность и управление**
- Отступы под "чёлку"/скруглённые углы (safe-area-inset), более гибкая
  сетка режимов (3 или 4 карточки без поломки вёрстки), уменьшенные
  элементы HUD на очень маленьких экранах.
- **Настроить расположение кнопок**: в Настройках — зеркалирование
  джойстика (слева/справа) и перетаскивание кнопок УДАР/OVERCLOCK/СМЕНА/
  ПАУЗА пальцем в отдельном редакторе, с сохранением.

## Как это тестировалось

Вся логика (прогрессия/скины/лига/турнир/формации) прогнана через node
с эмуляцией localStorage — числа сходятся: XP-кривая, стоимость
прокачки, разблокировки скинов, ISO-недели турнира (сверено с
эталонными датами), ранжирование лидерборда. Бэкенд-воркер прогнан
через фейковую in-memory D1 — сабмит/рейтинг/анти-спам работают верно.//
Весь фронтенд (все экраны, 6х6 формация, 400+ кадров симуляции матча,
победа/поражение/полное прохождение лиги, начисление наград, HUD-
редактор) прогнан через jsdom end-to-end без единой необработанной
ошибки. Живого браузера у меня нет, так что финальная визуальная
проверка (насколько красиво легло на конкретный экран телефона) — за
тобой, но структурно всё стыкуется.
